Locur

webdesign / webdevelopment
Health App

Support for the management of dental surgery.

Description

Support for the management of dental surgery. Locur allows users to control a dental surgery through the website. Doctor is able to look, search or save each visit through a calendar. Each visit can be written with informations about patient's teeth state (eg. surfaces, tooth actions, current state). Locur includes a list of all patients and a patient's profile page. Thanks to the dashboard, owner has an access to all necessary data(eg. costs/revanue) in one place.
